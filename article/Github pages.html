使用Github pages創建自己的Blog吧！


首先要先註冊Github的帳號與密碼！

1、Create a new repository



2、create branch：gh-pages

3、點選settings，設定預設branch

4、create new file

5、點選settings 在Options找到GitHub Pages

6、點選網址就可以看到您的Github pages囉！

恭喜您完成了第一個成就！趕快動手創建屬於自己的Blog吧~
