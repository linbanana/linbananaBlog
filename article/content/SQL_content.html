SQL指令應用


1.新增使用者帳號

        CREATE USER '使用者帳號'@'localhost' IDENTIFIED BY '使用者密碼';

2.將使用者帳號設定為ALL PRIVILEGES

        GRANT ALL PRIVILEGES on *.* to '使用者帳號'@'localhost';

3.建立資料庫

        CREATE DATABASE `資料庫名稱`

4.建立資料表

        CREATE TABLE `資料庫名稱`.`資料表名稱` (
          `欄位名稱` int(10) NOT NULL PRIMARY key AUTO_INCREMENT,
          `欄位名稱` varchar(10) NOT NULL,
          `欄位名稱` datetime
          ) ENGINE = InnoDB;

5.增加欄位、ENUM使用

        ALTER TABLE `資料表名稱` ADD `欄位名稱` ENUM('M','F','N','') NOT NULL DEFAULT 'N';

6.修改資料表結構

        ALTER TABLE `資料表名稱` CHANGE `舊欄位名稱` `新欄位名稱` varchar(10);

7.刪除資料表欄位

        ALTER TABLE `資料表名稱` DROP `欄位名稱`
