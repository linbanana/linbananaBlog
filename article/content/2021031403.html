SQL指令應用(合併資料表)


1.合併資料表tb1,tb2(UNION 垂直結合)

SELECT * FROM `tb1` UNION select * from `tb2`;

2.合併資料表tb1,tb2(JOIN 橫向結合)

1) INNER JOIN (內部連接) 為等值連接，必需指定等值連接的條件，而查詢結果只會返回符合連接條件的資料。
　SELECT tb1.`欄位名稱`, tb2.`欄位名稱` FROM tb1 INNER JOIN t2 ON tb1.`欄位名稱`=tb2.`欄位名稱`;
　or
  SELECT tb1.`欄位名稱`, tb2.`欄位名稱` FROM tb1 INNER JOIN t2 USING (`欄位名稱`);

2) LEFT JOIN 可以用來建立左外部連接，查詢的 SQL 敘述句 LEFT JOIN 左側資料表 (t1) 的所有記錄都會加入到查詢結果中，即使右側資料表 (t2) 中的連接欄位沒有符合的值也一樣。
  　SELECT tb1.`欄位名稱`, tb2.`欄位名稱` FROM tb1 LEFT JOIN t2 ON tb1.`欄位名稱`=tb2.`欄位名稱`;

3) RIGHT JOIN 相對於 LEFT JOIN，RIGHT JOIN 可以用來建立右外部連接，查詢的 SQL 敘述句 RIGHT JOIN 右側資料表 (table_name2) 的所有記錄都會加入到查詢結果中，即使左側資料表 (table_name1) 中的連接欄位沒有符合的值也一樣。
  　SELECT tb1.`欄位名稱`, tb2.`欄位名稱` FROM tb1 RIGHT JOIN t2 ON tb1.`欄位名稱`=tb2.`欄位名稱`;

4) FULL JOIN 即為 LEFT JOIN 與 RIGHT JOIN 的聯集，它會返回左右資料表中所有的紀錄，不論是否符合連接條件。
  　SELECT tb1.`欄位名稱`, tb2.`欄位名稱` FROM tb1 FULL JOIN t2 ON tb1.`欄位名稱`=tb2.`欄位名稱`;
